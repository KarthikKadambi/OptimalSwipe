OptimalSwipe is a high-performance, privacy-first web application designed to help you maximize your credit card rewards with zero data leakage. Every byte of your financial data stays inside your browser's encrypted vault.

## ‚ú® Features

- **100% Local & Private**: No servers, no tracking, no cloud databases. Your data belongs to you.
- **Biometric Security**: Protect your wallet with FaceID/TouchID (WebAuthn).
- **Universal Live Sync**: Automated file-linking sync for Desktop and an iOS Shortcut bridge for iPhone.
- **Smart Recommendation Engine**: Real-time advice on which card to use for any transaction.
- **PWA Standalone**: Install it as a native-feeling app on iOS and Android.
